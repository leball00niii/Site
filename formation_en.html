The CIEL Training Program

Cybersecurity, Computer Science and Electronics, Networks — A complete program to become a digital technology specialist.

🎓 What is the CIEL training program?

The CIEL program trains technicians capable of designing, installing and securing IT systems and communication networks. Students learn hardware configuration, server administration, application development and data security.

🎯 Why choose CIEL?

This program combines three essential fields: cybersecurity, computer science and electronics. It ensures high versatility and excellent employability in digital technology careers.

💡 Skills Developed

🔒 Cybersecurity

  • System security
  • Data protection
  • Ethical hacking

💻 Computer Science

  • Web development
  • Programming (Python, C, JS)
  • SQL databases

🌐 Networks

  • Network configuration
  • TCP/IP protocols
  • VPN and network security

⚡ Electronics

  • Microcontrollers (Arduino)
  • Raspberry Pi
  • IoT projects
